## 📊 Arquitetura Técnica:

### Arquivos Principais
```
index.html              - HTML + JavaScript inline (270 linhas)
style.css              - Estilos responsivos (1700+ linhas)
landing-page-data.json - 7 módulos com conteúdo
```

### Tecnologias
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Dados:** JSON estruturado
- **Hospedagem:** GitHub Pages (grátis, HTTPS automático)
- **Servidor Local:** Python http.server

### Funcionalidades de Código
- Hash-based routing (URLs dinâmicas)
- SessionStorage para autenticação
- LocalStorage para preferências de tema
- Fetch API para carregar dados
- Event listeners para interatividade
