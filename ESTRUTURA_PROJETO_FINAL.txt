────────────────────────────────────────────────────────────────────────────────
 FLUXO DE NAVEGAÇÃO
────────────────────────────────────────────────────────────────────────────────

1. INICIALIZAÇÃO
   ├── Página carrega
   ├── checkAuthentication() verifica sessionStorage
   ├── Se autenticado: mostra landing page
   └── Se não: mostra login screen

2. LOGIN
   ├── Usuário vê tela de login
   ├── Insere senha
   ├── submitPassword() valida
   ├── Se correto: sessionStorage['authenticated'] = 'true'
   └── Redireciona para landing page

3. NAVEGAÇÃO NA LANDING PAGE
   ├── Sidebar lista módulos
   ├── Clica em módulo: mostra tópicos
   ├── Clica em tópico: carrega dados do JSON
   ├── Renderiza slides
   ├── Next/Prev navega entre slides
   └── Hash URL reflete estado (#/mod-basic-1/bolsa/slide-4)

4. FUNCIONALIDADES ADICIONAIS
   ├── Dark/Light mode toggle
   ├── Logout button (sessionStorage.removeItem)
   ├── Breadcrumb navigation
   ├── Progress bar
   └── Responsive sidebar (mobile collapse)

5. LOGOUT
   ├── Clica logout() no header
   ├── sessionStorage['authenticated'] é removido
   ├── location.reload() reinicia
   └── Volta para login screen
