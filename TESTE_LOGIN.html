5. Fluxo de Autenticação

Sequência:
  1. Página carrega → checkAuthentication()
  2. Verifica sessionStorage.getItem('authenticated')
  3. Se NÃO autenticado → Mostra login screen
  4. Usuário digita senha → submitPassword()
  5. Se correto → sessionStorage.setItem('authenticated', 'true')
  6. Se correto → showLandingPage() → initialize()
  7. Landing page carrega normalmente
  8. Usuário clica "🚪 Sair" → logout()
  9. Limpa sessionStorage → Recarrega página
  10. Volta para tela de login
