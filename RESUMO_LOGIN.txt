FLUXO DE FUNCIONAMENTO
================================================================================

Quando a página é carregada:

1. DOMContentLoaded dispara
   ↓
2. Função checkAuthentication() é chamada
   ├─ Verifica: sessionStorage.getItem('authenticated') === 'true'?
   │
   ├─ SIM → Autenticado
   │  ├─ Oculta div#login-screen (display: none)
   │  ├─ Mostra div#slideContainer (display: block)
   │  ├─ Chama initialize() para carregar landing page
   │  └─ Usuário vê a plataforma normalmente
   │
   └─ NÃO → Não autenticado
      ├─ Mostra div#login-screen (display: flex)
      ├─ Oculta div#slideContainer (display: none)
      ├─ Foca no input de senha
      └─ Espera usuário digitar senha

3. Usuário digita senha e clica Entrar (ou pressiona Enter)
   ↓
4. Função submitPassword() é chamada
   ├─ Lê valor de document.getElementById('passwordInput')
   ├─ Compara com CORRECT_PASSWORD = 'AndreTrader'
   │
   ├─ SIM → Senha correta
   │  ├─ sessionStorage.setItem('authenticated', 'true')
   │  ├─ Chama showLandingPage()
   │  ├─ Oculta login screen
   │  ├─ Mostra landing page
   │  └─ Inicializa aplicação
   │
   └─ NÃO → Senha errada
      ├─ Exibe: "❌ Senha incorreta. Tente novamente."
      ├─ Limpa o campo de entrada
      └─ Foca no input novamente

5. Usuário clica botão "🚪 Sair" no header
   ↓
6. Função logout() é chamada
   ├─ sessionStorage.removeItem('authenticated')
   ├─ location.reload() - Recarrega a página
   └─ Volta para tela de login

SEGURANÇA
================================================================================

✅ Pontos Positivos:
   • sessionStorage não persiste entre abas/sessões
   • Sem armazenamento permanente de autenticação
   • Sem cookies (mais seguro)

SUPORTE E TROUBLESHOOTING
================================================================================

Problema: "A tela de login não aparece"
Solução:
  1. Verifique se index.html foi aberto no navegador
  2. Pressione F5 para recarregar
  3. Limpe o cache (Ctrl+Shift+Delete)
  4. Teste em outro navegador

Problema: "Senha não funciona"
Solução:
  1. Verifique se digitou exatamente: AndreTrader
  2. Atenção: A maiúscula é importante!
  3. Sem espaços no início ou fim
  4. Verifique o console (F12) para erros

Problema: "Botão Logout não aparece"
Solução:
  1. Recarregue a página (F5)
  2. Verifique se está autenticado (sessionStorage)
  3. Abra DevTools (F12) e execute:
     console.log(sessionStorage.getItem('authenticated'))
  4. Se retornar null, você não está autenticado

Problema: "Sempre pede login"
Solução:
  1. Isso é normal — sessionStorage não persiste entre abas
  2. Fechar o navegador limpa a autenticação
  3. Para persistir entre sessões, seria necessário usar localStorage
     (menos seguro) ou backend com cookies

Para mais detalhes, consulte: LOGIN_IMPLEMENTATION.md
